# WTW Task Application Challenge

## Description
This application was designed as an interview challenge for a position at Willis Towers Watson for a Front End Software Developer. A user can add, update, and delete tasks stored on a SQL Server.

## Languages/Packages
* React
* C#
* HTML
* CSS
* Bootstrap
* SQL Server

## Installation and Usage
Clone repo to local machine and run dotnet run to run the application locally.
